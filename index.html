Git & GitHub Learning

Git Commands

Setup

  1. git config --global user.name "username"
  2. git config --global user.email "email@example.com"

Push Code to GitHub

  1. git init
  2. git add <filename>
  3. git status
  4. git commit -m "<message>"
  5. git branch -M main
  6. git remote add origin <URL>
  7. git push -u origin main
